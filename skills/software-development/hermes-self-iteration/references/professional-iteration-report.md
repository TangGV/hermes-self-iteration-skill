# Professional Self-Iteration Report Template

Use this template when a self-iteration task is large, ambiguous, or user-facing enough that a short final summary would hide important decisions.

| 入口 | 文件 / URL / 服务 / workflow |
| 关键依赖 | ... |
| 约束 | 用户偏好、AGENTS、生产安全、凭证边界 |

- 体验/审美（如适用）：...

### 3. 问题分级

| 级别 | 问题 | 证据 | 处理 |

| P0 | ... | ... | ... |
| P1 | ... | ... | ... |
